## 新人须知

湖大古典服是由 Minecraft 岳麓幻境社开设的 `Paper` 端类原版 Minecraft 服务器，主要玩法为 **文明模拟**，服务端版本 `1.21.4`.

我们欢迎大家互相帮助, 交流与讨论.

## 服务器规则

禁止生电玩法.

## 入服指南

> **征集报错信息**
>
> 如果你在加入服务器过程中遇到了错误, 欢迎在群里发送截图, 崩溃报告等, 我们会维护一个服务器连接故障修复参考文档.

1. 客户端: Java `1.21.4` 版本, 无指定模组需求.
2. 登录方式: 仅支持正版验证, 请使用正版账号登录启动客户端.
3. 加入服务器: 服务器地址详见内部文档.

网页地图: [mc.hmomc.cn:25571](https://mc.hmomc.cn:25571)

> **遇到故障?**
>
> 如果你遇到无法连接服务器, 卡顿, 掉线, 打不开网页地图的情况, 可以参考这篇文档尝试解决. 如果仍为解决, 欢迎到群里提问.

## 服务器设置

- 难度: 困难
- 保留物品栏: 不开启
- 生物破坏: 开启

## 服务端插件说明

### 插件列表

```
bluemap-5.7-spigot // 网页地图
KingdomsX-1.17.10-BETA // 王国系统
LuckPerms-Bukkit-5.4.145 // 权限管理
QuickShop-Hikari-6.2.0.9-RELEASE-1 // 商店
TNE-Paper-0.1.3.5-Release-1 // 经济插件
spark-1.10.119-bukkit // 分析性能
VaultUnlocked-2.10.0 // 经济插件
```

### KingdomsX 王国系统

- `/k language 汉语` 切换汉语.
- `/k` 查看帮助.

### QuickShop Hikari 简易商店

放出一个箱子, 拿着你想出售(或者收购)的物品, 左键单击, 按提示操作即可.
